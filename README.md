<h1> Receipt Splitter</h1>
<h2> An app where users can input/track details for splitting receipt bills</h2>

The user can input Items, Price, Person Responsible for Item, Tax, Tip. The user is also given the option of whether to split tax and tip proportionally to 
their share of the bill or to evenly split it between themselves. The user can also track whether the person responsible paid their share of the bill.  
  
  App is still currently under development and is built with Swift to be compatible on all iPhone 15s and above.
